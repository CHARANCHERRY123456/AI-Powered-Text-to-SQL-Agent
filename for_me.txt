"This project is an AI-powered Natural Language to SQL Query System. It allows users to ask database questions in plain English, such as 'Who is the university topper?' or 'Show students with CGPA above 9'. The system uses Google's Gemini LLM along with LangChain and LangGraph to understand the user's query, automatically generate SQL statements, execute them on a PostgreSQL database, and return human-readable answers. It uses a ReAct Agent architecture, which enables the AI to inspect database schemas, identify relevant tables, generate correct SQL queries, execute them safely, and provide results without requiring users to know SQL."

Flow:

User Question → Gemini LLM → LangChain SQL Agent → Database Schema Analysis → SQL Query Generation → PostgreSQL Execution → Human Readable Response
